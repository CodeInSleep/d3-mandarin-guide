教學手冊簡介

本書不會解釋基本的html, css 跟javascript 功能，不會呈現過於專業的高難度技術，更不會讓你一天就變成D3專家。這個新手教學適合有基本html, css跟javascript 技術又想學最新D3技術的工程師。此書將透過最有效的實戰演練教您D3的入門技術，讓您能輕鬆越過一開始的新手學習障礙(Learning curve)，再獨自專研D3或將其應用在自己的領域。
此教學將先講解基本的D3概念，之後再用實例演練來進一步教學如何使用D3。
本書為Open source。在讀完之後如果您覺得受益良多，想要貢獻或合作我們都非常歡迎。
Github: d3-mandarin-guide
